**Preguntas normales:** 5 = máxima salud, 1 = mínima salud
**Preguntas invertidas (🔄):** Se recalculan como `6 - valor`. Ejemplo: si contestas "Siempre" (5), se convierte a 1. Esto es porque responder "Siempre" a una pregunta negativa indica un problema.

## SISTEMA DE PUNTUACIÓN Y CLASIFICACIÓN

### Cálculo por área

```
Para cada área:
  1. Tomar todas las preguntas del área
  2. Si inverted = true → puntaje = 6 - respuesta
  3. Si inverted = false → puntaje = respuesta tal cual
  4. Score del área = promedio de los puntajes ajustados (1-5)
  5. Porcentaje = ((score - 1) / 4) × 100
```

### Niveles por área

| Score | Nivel | Interpretación |
|---|---|---|
| ≤ 1.5 | Muy bajo | Área en crisis |
| 1.6 – 2.5 | Bajo | Área preocupante |
| 2.6 – 3.5 | Medio | Área funcional con oportunidad de mejora |
| 3.6 – 4.2 | Alto | Área saludable |
| > 4.2 | Muy alto | Área fortaleza |

### Resultado general (promedio de las 8 áreas)

| Promedio | Diagnóstico | Etiqueta |
|---|---|---|
| ≥ 3.8 | 🟢 Relación Sólida | "Existe una base emocional estable" |

| 2.2 – 2.99 | 🔴 Relación en Riesgo | "Los conflictos acumulados están afectando seriamente" |
| < 2.2 | ⚫ Relación Crítica | "Los patrones actuales pueden llevar a ruptura" |
